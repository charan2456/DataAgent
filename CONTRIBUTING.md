# Contributing to Data Agent

Thank you for your interest in contributing to Data Agent!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/charan2456/data-agent.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Commit: `git commit -m 'Add some feature'`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Development Setup

1. Follow the installation instructions in the main [README.md](./README.md)
2. Ensure MongoDB and Redis are running
3. Set up your environment variables (see README.md)
4. Run backend: `cd backend && python main.py`
5. Run frontend: `cd frontend && npm run dev`

## Code Style

- **Python**: Follow PEP 8
- **TypeScript/JavaScript**: Follow ESLint rules
- **Commit messages**: Use clear, descriptive messages

## Testing

Before submitting:
- Test locally
- Ensure no console errors
- Check that existing functionality works

## Pull Request Process

1. Update README.md if needed
2. Add tests if applicable
3. Ensure all checks pass
4. Request review from maintainers

## Questions?

Open an issue or start a discussion!
